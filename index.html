Tags utilizadas nas aulas

  • Os elementos HTML (h1–h6) representam seis níveis de título de seção. h1 é o nível da seção mais alto e h6 é o mais baixo.

    • (Voltar)

  • O Elemento HTML (blockquote) (ou Elemento HTML de citação de bloco) indica que o texto incluído é uma longa citação.

    • (Voltar)

  • O elemento HTML (li) (ou a Lista dos Itens de um elemento HTML) é usado para representar um item que faz parte de uma lista. Este item deve estar contido em um elemento pai: uma lista ordenada ( ol ), uma lista desordenada ( ul)

    (Voltar)

  • O elemento HTML (p) representa um parágrafo. Em mídias visuais, os parágrafos são representados como blocos recuados de texto com a primeira letra avançada e separados por linhas em branco. Já em HTML, os parágrafos são usados ​​para agrupar conteúdos relacionados de qualquer tipo, como imagens e campos de um formulário.

    • (Voltar)

  • O elemento HTML (a) em HTML (ou elemento âncora), com o atributo hrefcria-se uma hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL. O conteúdo de cada um a precisará indicar o destino do link.

    • (Voltar)

  • O elemento HTML quebra-de-linha (br) produz uma quebra de linha em um texto (carriage-return).É útil para escrever poemas ou um endereço, onde a divisão de linha é significativa.

    • (Voltar)

  • O elemento HTML (abbr)_(ou Elemento de Abreviação HTML) representa uma abreviação e opcionalmente fornece uma descrição completa para ela. Se presente, o atributo titledeve conter a descrição completa e apenas ela.

    • (Voltar)

  • O elemento HTML (mark),representa um trecho de destaque em um texto, por exemplo, uma sequência de texto marcada como referência, devido à sua relevância em um contexto particular. Por exemplo, pode ser utilizado em uma página que mostra os resultados de uma busca onde todas as instâncias da palavra pesquisadas receberam destaque.

    • (Voltar)

  • O elemento HTML (sub) Especifica o texto embutido que deve ser exibido como subscrito apenas por motivos tipográficos. Os subscritos normalmente são renderizados com uma linha de base reduzida usando texto menor.

    • (Voltar)

  • O elemento HTML (sup) Especifica o texto embutido que deve ser exibido como sobrescrito apenas por motivos tipográficos. Os sobrescritos geralmente são renderizados com uma linha de base elevada usando texto menor.

    • (Voltar)

  • O elemento HTML (del) (ou Elemento HTML de Texto Excluído ) representa uma parte do texto que foi a restauração de um documento. Este elemento é (não necessariamente) renderizado pelos navegadores como uma linha entre o texto.

    • (Voltar)

  • O elemento HTML (font) Define o tamanho da fonte, cor e face do seu conteúdo.

    • (Voltar)
